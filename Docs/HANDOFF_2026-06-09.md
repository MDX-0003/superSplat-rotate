## 一、工作概述

本次开发围绕 **cameras.json 导入流程优化** 展开：

1. **导入 cameras.json 自动填充时间轴关键帧** — 不再需要手动逐帧添加
2. **导入时自动设置时间轴总帧数** — 省去手动调 UI
3. **批量导出速度优化** — `lockedRenderMode` + 显式 sort 替代 `forceRender`
4. **迁移文档** — 记录了 origin 仓库畸变校正方案的迁移方案

### 2.2 `src/file-handler.ts` (+14/-4 行)

**2.2a `loadCameraPoses`（cameras.json 导入）**：

在原有 `addImportedPose` 事件基础上，新增三件事：

| 顺序 | 操作 | 说明 |
|------|------|------|
| 1 | `events.fire('camera.clearPoses')` | 导入前清空时间轴旧关键帧 |
| 2 | `events.fire('camera.addPose', payload)` | 每条位姿同时写入时间轴（复用同一 payload，安全：`clonePose()` + `addPose()` 各自深克隆） |
| 3 | `events.fire('timeline.setFrames', json.length)` | 导入后自动设置总帧数 |

位置：[file-handler.ts:188-234](src/file-handler.ts#L188-L234)

**2.2b `loadImagesTxt`（COLMAP images.txt 导入）**：

同样的三件事，总帧数 = `poses.length`。

位置：[file-handler.ts:276-299](src/file-handler.ts#L276-L299)

### 2.3 `src/render.ts`（批量导出速度优化）

**Export All 改为 `lockedRenderMode` + 显式 sort**，对齐视频渲染路径的性能：

| 改动 | 旧 | 新 |
|------|-----|-----|
| 渲染控制 | `scene.forceRender = true` | `scene.lockedRender = true` |
| 场景 diff | 每帧 `pack() → compare()` | 跳过（lockedRenderMode） |
| 排序触发 | 依赖 `camera.setPose` 间接触发 | 显式 `instance.sort(camera)` |
| 排序等待 | 循环内每次重建 Promise | 提取 `sortAndWait` 复用 |
| visibleSplats | 循环内每次查询 | 提到循环外 |

位置：[render.ts:186-277](src/render.ts#L186-L277)

## 四、glTF → cameras.json 工具

`tills/gltf_to_cameras_json.py` — 从 glTF 相机动画生成 cameras.json，纯 Python 标准库，零外部依赖。

**用法**：

```bash
python tills/gltf_to_cameras_json.py SequenceData/01/CamSqe.gltf -o SequenceData/01/cameras.json
```

**已知问题**：脚本将同一个 FOV 代入 width 和 height 两条公式，导致 `fx ≠ fy`（`fx/fy = width/height`）。对于正方形像素相机，fx 应等于 fy。需改脚本第 462 行 `fx = fy`。

## 六、项目当前架构要点

### 相机位姿两条数据流

```
cameras.json 导入
  ├─→ camera.addImportedPose → importedPoses[]  (GT 相机列表，View Panel 预览)
  └─→ camera.addPose → CameraAnimTrack.poses[]  (时间轴关键帧，Spline 插值 + 视频渲染)
```

两个列表独立存储、独立消费。导入时同时填充。

### 关键事件

| 事件 | 用途 |
|------|------|
| `camera.clearImportedPoses` | 清空 GT 列表 |
| `camera.clearPoses` | 清空时间轴 track（本次新增） |
| `camera.addImportedPose` | 添加到 GT 列表 |
| `camera.addPose` | 添加到时间轴 track |
| `timeline.setFrames` | 设置总帧数 |
